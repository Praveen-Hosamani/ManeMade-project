# ManeMade Project

A full-stack application with React JS, Spring Boot, and PostgreSQL.

## Structure
- `frontend/`: React JS application (Vite)
- `backend/`: Spring Boot application (Maven)

## Getting Started

### Frontend
1. Navigate to `frontend/`
2. Run `npm install`
3. Run `npm run dev`

### Backend
1. Navigate to `backend/`
2. Configure `src/main/resources/application.properties`
3. Run with Maven or your IDE.
